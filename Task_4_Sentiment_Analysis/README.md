Sentiment Analysis using Machine Learning

Project Overview

This project performs sentiment analysis on social media text data to classify sentiments as Positive, Neutral, or Negative.

Objective

To build a machine learning model capable of understanding public opinion from textual data.

Tools and Libraries Used

- Python
- Pandas
- Scikit-learn
- TF-IDF Vectorizer
- Matplotlib
- Seaborn
- Kaggle
- GitHub

Dataset Features

- Year
- Month
- Day
- Time of Tweet
- Text
- Sentiment
- Platform

Steps Performed

1. Data Loading and Exploration
2. Sentiment Distribution Analysis
3. Platform Analysis
4. Text Vectorization using TF-IDF
5. Label Encoding
6. Train-Test Split
7. Model Training using Multinomial Naive Bayes
8. Model Evaluation
9. Sentiment Prediction on Custom Text

Results

- Successfully classified text into Positive, Neutral, and Negative sentiments.
- Achieved approximately 62% accuracy.
- Generated confusion matrix and classification report.

Conclusion

Sentiment Analysis helps organizations understand customer opinions and public sentiment from large volumes of text data.
